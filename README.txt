Chicken Road - Kivy Version (Android Ready)

How to Run on Android:
1. Install Python, Kivy, and Buildozer on Linux.
2. In terminal, run:
    buildozer init
3. Edit buildozer.spec:
    - package.name = ChickenRoad
    - requirements = python3,kivy
4. Then run:
    buildozer -v android debug
5. APK will be in the bin/ folder.

Use touch to move the chicken in the game.
Enjoy!
